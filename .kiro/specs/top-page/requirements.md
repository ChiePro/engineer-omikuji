# トップページ要件定義

## 概要
エンジニア向けおみくじWebサービスのトップページ。ユーザーが一目でサービスを理解し、おみくじを引きたくなる体験を提供する。

## 背景と目的

### 背景
- エンジニアの日常に楽しみと気づきを提供するエンターテインメントサービス
- 説明文ではなく視覚的に理解させる必要がある
- おみくじを引くというシンプルなアクションに導く必要がある

### 目的
1. サービスの内容を視覚的・直感的に伝える
2. おみくじを引く動機を醸成する
3. スムーズな導線でおみくじ選択へ誘導する

## 機能要件

### FR-TOP-001: ビジュアルアイデンティティ
#### 神社×テクノロジーの融合表現
- 背景に神社要素（鳥居、社殿、お賽銭箱）を配置
- エンジニア要素（ターミナル風UI、コードスニペット）を融合
- 和風の落ち着きとテック感の調和

#### キャッチコピーとメッセージング
- メインコピー：「エンジニアの運命を占う」
- サブコピー：「今日のコーディング運は？」
- 視覚的に目立つ配置で興味を引く

### FR-TOP-002: おみくじ種類の表示
#### おみくじカードUI
各おみくじを以下の形式でカード表示：

1. **エンジニア運勢**
   - アイコン：⚡ または `<code/>` 記号
   - 説明：「今日のコーディングを占う」
   - カラー：電気的な青

2. **技術選定おみくじ**
   - アイコン：🎲 または技術スタックシンボル
   - 説明：「次に学ぶ技術を決める」
   - カラー：多彩な色

3. **デバッグ運**
   - アイコン：🐛 またはバグシンボル
   - 説明：「バグ解決のヒントを得る」
   - カラー：注意を引く赤系

4. **コードレビュー運**
   - アイコン：👀 またはレビューシンボル
   - 説明：「レビューの結果を予想」
   - カラー：チェックの緑系

5. **デプロイ運**
   - アイコン：🚀 またはロケットシンボル
   - 説明：「デプロイの成功を占う」
   - カラー：推進力のあるオレンジ

#### カードインタラクション
- ホバー時：浮き上がるアニメーション
- クリック可能エリア：カード全体
- 選択時：軽い押し込みエフェクト

### FR-TOP-003: 動機付けシステムの可視化
#### レアリティプレビュー
- 画面上部または側面にレアリティ階層を表示
- Common → Rare → Epic → Legendary の順で視覚化
- キラキラエフェクトで希少性を演出
- 「レジェンダリーが出るかも？」のテキスト表示

#### お賽銭システムの示唆
- お賽銭箱のビジュアルを配置
- 「お賽銭で運気UP」の控えめな表示
- 詳細は別画面で説明（トップページでは示唆のみ）

### FR-TOP-004: アクション誘導
#### CTA（Call to Action）設計
- 各カードに「このおみくじを引く」ボタン
- プライマリボタンのデザイン
- ホバー時：輝きのアニメーション
- 押下時：波紋エフェクト

#### 画面遷移
- カード選択 → おみくじ画面へ遷移
- スムーズなページトランジション
- 選択したおみくじ種類を次画面へ引き継ぎ

## 非機能要件

### NFR-TOP-001: パフォーマンス要件
- First Contentful Paint (FCP): < 1.0秒
- Time to Interactive (TTI): < 2.5秒
- Cumulative Layout Shift (CLS): < 0.1
- アニメーション: 60fps維持

### NFR-TOP-002: アクセシビリティ要件
- WCAG 2.1 Level AA準拠
- キーボードナビゲーション完全対応
- スクリーンリーダー対応（適切なARIAラベル）
- 色覚多様性への配慮（色だけに依存しない情報伝達）

### NFR-TOP-003: 互換性要件
- ブラウザ：Chrome, Firefox, Safari, Edge（各最新版）
- デバイス：デスクトップ、タブレット、モバイル
- レスポンシブデザイン（320px〜）

## 制約事項
- 初期実装ではユーザー認証なし
- データの永続化なし（セッションストレージのみ）
- 外部APIへの依存なし

## 前提条件
- Next.js 14+ (App Router) を使用
- Tailwind CSSでスタイリング
- Framer Motionでアニメーション実装
- TypeScript strictモードで開発

## 受け入れ基準
1. ページ表示時、3秒以内に全要素が表示される
2. 5種類のおみくじがカード形式で表示される
3. 各カードにホバー・クリックアニメーションが動作する
4. レアリティシステムが視覚的に理解できる
5. お賽銭箱が表示され、システムの存在が示唆される
6. カード選択でおみくじ画面へ遷移する
7. モバイル・デスクトップ両方で適切に表示される
